## Development

We use [yarn](https://yarnpkg.com) and [parceljs](https://parceljs.org/).
Run parcel for development:

```
yarn start
```

### Tests

We currently use [ava](https://github.com/avajs/ava) as test runner.
Run the tests with

```
yarn test
```
